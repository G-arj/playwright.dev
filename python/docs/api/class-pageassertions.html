PageAssertions

The PageAssertions class provides assertion methods that can be used to make assertions about the Page state in the tests. A new instance of LocatorAssertions is created by calling playwright_assertions.expect(page):

  • page_assertions.not_to_have_title(title_or_reg_exp, **kwargs)
  • page_assertions.not_to_have_url(url_or_reg_exp, **kwargs)
  • page_assertions.to_have_title(title_or_reg_exp, **kwargs)
  • page_assertions.to_have_url(url_or_reg_exp, **kwargs)
