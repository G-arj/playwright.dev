WorkerInfo

WorkerInfo contains information about the worker that is running tests. It is available to test.beforeAll(hookFunction) and test.afterAll(hookFunction) hooks and worker-scoped fixtures.

  • TypeScript
  • JavaScript
import { test, expect } from '@playwright/test';
test.beforeAll(async ({ browserName }, workerInfo) => {  console.log(`Running ${browserName} in worker #${workerInfo.workerIndex}`);});
const { test, expect } = require('@playwright/test');
test.beforeAll(async ({ browserName }, workerInfo) => {  console.log(`Running ${browserName} in worker #${workerInfo.workerIndex}`);});
  • workerInfo.config
  • workerInfo.project
  • workerInfo.workerIndex

workerInfo.config#

  • type: <TestConfig>

Processed configuration from the configuration file.

workerInfo.project#

  • type: <TestProject>

Processed project configuration from the configuration file.

workerInfo.workerIndex#

  • type: <number>

The unique index of the worker process that is running the test. Also available as process.env.TEST_WORKER_INDEX. Learn more about parallelism and sharding with Playwright Test.
