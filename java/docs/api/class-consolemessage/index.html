ConsoleMessage

ConsoleMessage objects are dispatched by page via the Page.onConsoleMessage(handler) event.

  • ConsoleMessage.args()
  • ConsoleMessage.location()
  • ConsoleMessage.text()
  • ConsoleMessage.type()

ConsoleMessage.args()#

  • returns: <List<JSHandle>>

List of arguments passed to a console function call. See also Page.onConsoleMessage(handler).

ConsoleMessage.location()#

  • returns: <String>

URL of the resource followed by 0-based line and column numbers in the resource formatted as URL:line:column.

ConsoleMessage.text()#

  • returns: <String>

The text of the console message.

ConsoleMessage.type()#

  • returns: <String>

One of the following values: 'log', 'debug', 'info', 'error', 'warning', 'dir', 'dirxml', 'table', 'trace', 'clear', 'startGroup', 'startGroupCollapsed', 'endGroup', 'assert', 'profile', 'profileEnd', 'count', 'timeEnd'.
