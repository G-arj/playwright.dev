Mouse

The Mouse class operates in main-frame CSS pixels relative to the top-left corner of the viewport.

Every page object has its own Mouse, accessible with Page.mouse().

// Using â€˜page.mouseâ€™ to trace a 100x100 square.
page.mouse().move(0, 0);
page.mouse().down();
page.mouse().move(0, 100);
page.mouse().move(100, 100);
page.mouse().move(100, 0);
page.mouse().move(0, 0);
page.mouse().up();

Mouse.click(x, y[, options])#

  • x <double>
  • y <double>
  • options <Mouse.ClickOptions>
    • withButton <enum MouseButton { LEFT, RIGHT, MIDDLE }> Defaults to left.
    • withClickCount <int> defaults to 1. See UIEvent.detail.
    • withDelay <double> Time to wait between mousedown and mouseup in milliseconds. Defaults to 0.

Shortcut for Mouse.move(x, y[, options]), Mouse.down([options]), Mouse.up([options]).

Mouse.dblclick(x, y[, options])#

  • x <double>
  • y <double>
  • options <Mouse.DblclickOptions>
    • withButton <enum MouseButton { LEFT, RIGHT, MIDDLE }> Defaults to left.
    • withDelay <double> Time to wait between mousedown and mouseup in milliseconds. Defaults to 0.

Shortcut for Mouse.move(x, y[, options]), Mouse.down([options]), Mouse.up([options]), Mouse.down([options]) and Mouse.up([options]).

Mouse.down([options])#

  • options <Mouse.DownOptions>
    • withButton <enum MouseButton { LEFT, RIGHT, MIDDLE }> Defaults to left.
    • withClickCount <int> defaults to 1. See UIEvent.detail.

Dispatches a mousedown event.

Mouse.move(x, y[, options])#

  • x <double>
  • y <double>
  • options <Mouse.MoveOptions>
    • withSteps <int> defaults to 1. Sends intermediate mousemove events.

Dispatches a mousemove event.

Mouse.up([options])#

  • options <Mouse.UpOptions>
    • withButton <enum MouseButton { LEFT, RIGHT, MIDDLE }> Defaults to left.
    • withClickCount <int> defaults to 1. See UIEvent.detail.

Dispatches a mouseup event.
