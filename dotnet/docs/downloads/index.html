Downloads

note

For uploading files, see the uploading files section.

For every attachment downloaded by the page, event Page.Download event is emitted. If you create a browser context with the acceptDownloads set, all these attachments are going to be downloaded into a temporary folder. You can obtain the download url, file system path and payload stream using the Download object from the event.

You can specify where to persist downloaded files using the downloadsPath option in BrowserType.LaunchAsync(â€¦).

note
