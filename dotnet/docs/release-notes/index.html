Browser Versions#

  • Chromium 92.0.4498.0

New APIs#

  • support for async predicates across the API in methods such as Page.RunAndWaitForRequestAsync(action, urlOrPredicate, options) and others
  • new emulation devices: Galaxy S8, Galaxy S9+, Galaxy Tab S4, Pixel 3, Pixel 4
  • new methods:
    • Page.WaitForURLAsync(url, options) to await navigations to URL
    • Video.DeleteAsync() and Video.SaveAsAsync(path) to manage screen recording
  • new options:
    • screen option in the Browser.NewContextAsync(options) method to emulate window.screen dimensions
    • position option in Page.CheckAsync(selector, options) and Page.UncheckAsync(selector, options) methods
    • trial option to dry-run actions in Page.CheckAsync(selector, options), Page.UncheckAsync(selector, options), Page.ClickAsync(selector, options), Page.DblClickAsync(selector, options), Page.HoverAsync(selector, options) and Page.TapAsync(selector, options)

  • Chromium screenshots are fast on Mac & Windows.

Version 1.9#

  • Playwright Inspector is a new GUI tool to author and debug your tests.
    • Line-by-line debugging of your Playwright scripts, with play, pause and step-through.
    • Author new scripts by recording user actions.
    • Generate element selectors for your script by hovering over elements.
    • Set the PWDEBUG=1 environment variable to launch the Inspector
  • Pause script execution with Page.PauseAsync() in headed mode. Pausing the page launches Playwright Inspector for debugging.
  • New has-text pseudo-class for CSS selectors. :has-text("example") matches any element containing "example" somewhere inside, possibly in a child or a descendant element. See more examples.
  • Page dialogs are now auto-dismissed during execution, unless a listener for dialog event is configured. Learn more about this.
  • Playwright for Python is now stable with an idiomatic snake case API and pre-built Docker image to run tests in CI/CD.

Version 1.8#

  • Selecting elements based on layout with :left-of(), :right-of(), :above() and :below().
  • Playwright now includes command line interface, former playwright-cli.
  • Page.SelectOptionAsync(selector, values, options) now waits for the options to be present.
  • New methods to assert element state like Page.IsEditableAsync(selector, options).

Version 1.7#

  • New Java SDK: Playwright for Java is now on par with JavaScript, Python and C# bindings.
  • Browser storage API: New convenience APIs to save and load browser storage state (cookies, local storage) to simplify automation scenarios with authentication.
  • New CSS selectors: We heard your feedback for more flexible selectors and have revamped the selectors implementation. Playwright 1.7 introduces new CSS extensions and there's more coming soon.
  • New website: The docs website at playwright.dev has been updated and is now built with Docusaurus.
  • Support for Apple Silicon: Playwright browser binaries for WebKit and Chromium are now built for Apple Silicon.

New APIs#

  • BrowserContext.StorageStateAsync(options) to get current state for later reuse.
  • storageState option in Browser.NewContextAsync(options) and Browser.NewPageAsync(options) to setup browser context state.
